# 🎬 Movie Database App

A responsive React application that allows users to browse and explore movies using The Movie Database (TMDB) API. Users can view popular, top-rated, and upcoming movies, search for movies, and access detailed information including cast details.

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/movie-database.git
```

### Navigate into the project

```bash
cd movie-database
```

### Install dependencies

```bash
npm install
```

### Create a `.env` file

```env
VITE_API_KEY=YOUR_TMDB_API_KEY
```

Generate your API key from:

https://www.themoviedb.org/

### Start the development server

```bash
npm run dev
```

Open

```
http://localhost:5173
```
